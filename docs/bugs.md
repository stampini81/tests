# Bugs encontrados

Documente aqui qualquer bug identificado durante a execução dos testes.

## Exemplo de registro

- **Regra testada:** Menor de idade não pode ter receitas
- **Resultado esperado:** O sistema deve impedir o cadastro de receita para menor de idade
- **Resultado obtido:** O sistema permitiu o cadastro
- **Passos para reproduzir:**
  1. Cadastrar uma pessoa menor de idade
  2. Tentar cadastrar uma receita para essa pessoa
